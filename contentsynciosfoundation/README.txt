Required Dependencies
=====================

++FastPDFKit
This library must be checked out of GitHub and live relative to the project at ../ThirdParty/FastPdfKit.

From the ../ThirdParty directory run the following command-line:

git clone -b devel https://github.com/mobfarm/FastPdfKit.git.


NOTE:  At the time of writing it was necessary to pull changes from the devel branch (not Master) as enhancements for iPad Retina were not rolled into master yet.  Please check periodically and pull from master when ready.
